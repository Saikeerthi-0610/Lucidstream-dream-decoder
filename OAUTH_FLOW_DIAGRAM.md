## Provider-Specific Details

### Google OAuth
```
Authorize URL: https://accounts.google.com/o/oauth2/v2/auth
Token URL:     https://oauth2.googleapis.com/token
UserInfo URL:  https://www.googleapis.com/oauth2/v2/userinfo
Scopes:        openid, email, profile
Returns:       id, email, name, picture
```

### GitHub OAuth
```
Authorize URL: https://github.com/login/oauth/authorize
Token URL:     https://github.com/login/oauth/access_token
UserInfo URL:  https://api.github.com/user
               https://api.github.com/user/emails (for email)
Scopes:        user:email
Returns:       id, login, name, email
```

### Facebook OAuth
```
Authorize URL: https://www.facebook.com/v18.0/dialog/oauth
Token URL:     https://graph.facebook.com/v18.0/oauth/access_token
UserInfo URL:  https://graph.facebook.com/me?fields=id,name,email
Scopes:        email, public_profile
Returns:       id, name, email
```

### LinkedIn OAuth
```
Authorize URL: https://www.linkedin.com/oauth/v2/authorization
Token URL:     https://www.linkedin.com/oauth/v2/accessToken
UserInfo URL:  https://api.linkedin.com/v2/userinfo (OpenID Connect)
Scopes:        openid, profile, email
Returns:       sub (id), name, email
```
